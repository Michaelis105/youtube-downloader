# youtube-downloader
Simple video and audio downloader for YouTube sources.

# WARNING

1. Downloading YouTube video or audio data may be a violation of Terms of Services.
2. The writer is NOT responsible for any harm or damages as a result of using this program.
3. USE AT YOUR OWN RISK.

# Setup

Tested using Python 3.12.4

```
pip install pytubefix moviepy
```

# Usage

```
python main.py
    --url https://www.youtube.com/watch?v=blah
    --download_path C:\path\to\directory\
    --video_filename foobar
    --audio_filename foobar
```
